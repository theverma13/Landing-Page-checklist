# Landing-Page-checklist

#### Landing Page

**1) Headline**  
- [ ] Main Headline  
- [ ] Subheadline

**2) Hero Section**  
- [ ] Visuals (Image/Video)  
- [ ] Call-to-Action (CTA) Button  
- [ ] Supporting Text

**3) Value Proposition**  
- [ ] Benefits  
- [ ] Unique Selling Points (USPs)  
- [ ] Trust Signals (awards, certifications)

**4) Features/Solutions**  
- [ ] Feature List  
- [ ] Icons/Visuals  
- [ ] Short Descriptions

**5) Social Proof**  
- [ ] Testimonials  
- [ ] Case Studies  
- [ ] Logos of Clients/Partners  
- [ ] Reviews/Ratings

**6) Call-to-Action (CTA)**  
- [ ] Primary CTA  
- [ ] Secondary CTA  
- [ ] CTA Placement

**7) Benefits/Use Cases**  
- [ ] Problem-Solution Framework  
- [ ] Scenarios/Examples

**8) Pricing Section**  
- [ ] Pricing Tiers  
- [ ] Comparison Table  
- [ ] Discounts/Offers

**9) FAQ Section**  
- [ ] Common Questions  
- [ ] Concise Answers

**10) Footer**  
- [ ] Contact Info  
- [ ] Social Links  
- [ ] Privacy Policy/Terms
